# SIMRS Khanza Web — Modul Kebidanan & Kecantikan

Modul web PHP native pelengkap SIMRS Khanza (Java), khusus untuk klinik
kebidanan & kecantikan. Terhubung ke database `sik` bawaan Khanza
**tanpa mengubah skema sama sekali**.

## Dokumentasi
- `docs/PLAN-SIMRS-KHANZA-WEB-KEBIDANAN.md` — rencana lengkap & peta tabel database
- `docs/KEPUTUSAN-TEKNIS.md` — log keputusan teknis & progress (BACA INI dulu saat lanjut sesi baru)

## Quick start
1. Edit `config/koneksi.php` — isi kredensial DB `sik` server RSU Al-Arif.
2. Buka `test-koneksi.php` di browser untuk verifikasi koneksi.
3. Buka `preview-tema.html` untuk lihat tema warna.
